Online Store

This Java code implements an application to manage a product database using the ProductDatabase class, which utilizes a map to store products with operations to add, update, and delete products, as well as to calculate statistics. The Product class defines the properties of each product. Students CRUD

This Python code implements a system to manage students, subjects, and grades, with various functionalities accessible through an interactive menu. Student, subject, and grade data are stored in lists. The main menu offers options to add students, subjects, and grades, view the average grade per student, view the highest grade of a student, view the lowest grade per subject, and list students. Each menu option calls a specific function that performs the corresponding operations, including input validation and updating the data lists. The program continues to run until the user chooses to exit.

Library Management

The code implements a management system for a library with interactive menus in the console, allowing the administration of clients, books, sales, and statistics. It provides functions to add, list, update, search, and delete records of clients, books, and sales, as well as generate statistics such as total sales by ISBN, most and least sold books, and clients with the highest purchases.
